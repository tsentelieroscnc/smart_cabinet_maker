# Smart Cabinet Maker Pro (v2.6.0)

A high-precision parametric cabinet configurator for SketchUp, designed for professional woodworkers and CNC production.

## 🚀 Features
- **Parametric Engine:** Real-time generation of cabinet bodies, interiors, and fronts.
- **Dynamic Joinery:** Automatic 3D drilling for Dowels, Screws, and Minifix based on construction rules.
- **Back Panel Logic:** Support for Nailed, Grooved, and Hybrid (Grooved sides + Nailed top/bottom) construction.
- **Professional UI:** Compact Dark-Mode interface with localized Greek labels and Property Grid layout.
- **Preset System:** Save and load configuration templates for different production standards.

## 🛠 Installation
1. Download the repository as a ZIP or clone it.
2. Copy the `smart_cabinet_maker` folder to your SketchUp Plugins directory.
3. Restart SketchUp or load the `main.rb` via Ruby Console.

## 📁 Project Structure
- `main.rb`: Core Ruby logic and geometry generation.
- `ui/`: HTML/CSS/JS files for the HtmlDialog interface.
- `presets.json`: Storage for your custom cabinet presets.
